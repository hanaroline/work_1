# 신길파크자이 매수 홍보 브로슈어 (1페이지)

서울 영등포구 **신길파크자이**(신길뉴타운 8구역, GS건설 자이) 매수 관점의 장점을 극대화한
**단일 HTML 1페이지 브로슈어**입니다. 별도 빌드 없이 `index.html` 하나로 동작하며,
브라우저에서 열어 A4 세로로 인쇄/PDF 저장할 수 있습니다.

## 구성
1. **히어로** — 단지 정체성(8구역·GS건설·641세대·2020.12 입주) + 스카이라인 SVG
2. **핵심 스탯 스트립** — 641세대 / 신풍역 7분 / 여의도 2정거장 / 분양 대비 약 2.5배
3. **매수 강점 4카드** — 입지·교통 / 브랜드·상품성 / 교육·생활 / 투자가치
4. **신길뉴타운 심층 분석** — 주변 대장 단지 84㎡ 시세 비교 + 개발 호재(신안산선·신길2구역)
5. **가치 상승** — 분양가→현재 시세 추이 스파크라인
6. **입지·생활권** 요약 + 고지 푸터

## 디자인
- 딥 미드나잇 네이비 + 에메랄드 + 골드의 보석 톤(jewel-tone) 럭셔리 부동산 브로슈어
- 반응형 레이아웃, `@media print` A4 대응, `prefers-reduced-motion` 존중
- 외부 리소스 없이 자체 완결(인라인 SVG·CSS만 사용)

## 데이터 기준 및 고지
공개 실거래·언론 보도 기반의 **홍보·참고용** 자료(작성 기준 2026.07)입니다.
시세·분양가·상승 배수는 근사치이며 시점·평형·층·향에 따라 달라질 수 있고,
개발 호재는 추진 상황에 따라 변동될 수 있습니다. 투자 권유·수익 보장이 아니며,
계약 전 국토교통부 실거래가·현장 확인을 권장합니다.
